How to use the logs (quick)

  1. Fill the log on the page. For cleaning, tick the day; for temp logs, type values.
  2. Press Save every time you update. This saves to the browser (local device).
  3. At the end of each month, go to Archive → Save Month to download a single JSON file.
  4. Keep the JSON in the Archive folder (Google Drive / USB). That’s your official record.
  5. To print an old month: go to Archive → Load, select a JSON file, choose a record, then print.
  6. At year end you may Bundle Year to create a single file containing all 12 months.

Note: This version stores daily work locally (no cloud). Archiving downloads your data so you can back it up and share.
